⚙️ Zoho Sheet Connection Setup

1Create a Zoho API Client

Open api-console.zoho.com → Add Client → Choose "Client-based Application" → Set Authorized Redirect URI to this page's URL (shown below) → Copy the Client ID.

2Create your Zoho Sheet

In Zoho Sheet, create a new workbook. On row 1, add these exact headers (copy-paste):
Date | Rep | School | Type | Status | Contact | Description | Next Date | Next Action | Record ID
Name the sheet tab: School Visits. Then copy the Workbook ID from the URL:
sheet.zoho.com/sheet/open/{WORKBOOK_ID}/sheets/…

3Share with team

Share the Zoho Sheet (Edit access) with all sales reps. Share this HTML file in your Zoho WorkDrive — everyone opens it and connects with their own Zoho account.
